<summary>⚙️ Click to set up the Snake animation</summary>

Create `.github/workflows/snake.yml` in your profile repo:

```yaml
name: Generate Snake
on:
  schedule:
    - cron: "0 0 * * *"
  workflow_dispatch:
jobs:
  generate:
    runs-on: ubuntu-latest
    steps:
      - uses: Platane/snk@v3
        with:
          github_user_name: ${{ github.repository_owner }}
          outputs: |
            dist/github-contribution-grid-snake.svg
            dist/github-contribution-grid-snake-dark.svg?palette=github-dark
      - uses: crazy-max/ghaction-github-pages@v3
        with:
          target_branch: output
          build_dir: dist
        env:
          GITHUB_TOKEN: ${{ secrets.GITHUB_TOKEN }}
```

### ♟️ Play Chess With Me!

> Visitors can play chess against me right here! Click a piece to make your move — it opens a GitHub issue which triggers an Action to update the board. Let's play! 🎮

<div align="center">

<!-- Chess board auto-generated by readme-chess GitHub Action -->
![Chess Board](https://github.com/vaibhav-buildz/vaibhav-buildz/blob/main/chess.svg?raw=true)

</div>

> **⚙️ Setup Instructions** — expand below to activate the Chess game on your profile!

<details>
<summary>📋 Click to set up Chess</summary>

**Step 1** — Create `chess.svg` in the root of your profile repo with any placeholder SVG (the Action will overwrite it).

**Step 2** — Create `.github/workflows/chess.yml`:

```yaml
name: Update Chess Game
on:
  issues:
    types: [opened]
  workflow_dispatch:

jobs:
  update:
    runs-on: ubuntu-latest
    if: startsWith(github.event.issue.title, 'chess')
    steps:
      - uses: actions/checkout@v3

      - name: Play chess move
        uses: marcizhu/readme-chess@master
        with:
          github-token: ${{ secrets.GITHUB_TOKEN }}
          issue-number: ${{ github.event.issue.number }}

      - name: Commit changes
        run: |
          git config --local user.email "action@github.com"
          git config --local user.name "GitHub Action"
          git add chess.svg
          git diff --quiet && git diff --staged --quiet || git commit -m "Update chess board"
          git push
```

**Step 3** — Run the workflow manually once (`Actions → Update Chess Game → Run workflow`) to generate the initial board.

**Step 4** — Update the image link in the README to point to your `chess.svg`.

Now anyone who visits your profile can click a piece, which opens an issue like `chess: e2e4` — the Action processes the move and updates the board automatically! ♟️

</details>
